# DSA-2 Case Studies (CO1–CO6)

## Overview

This repository contains Java implementations and case study solutions developed as part of the **Data Structures and Algorithms – II (DSA-2)** course.

The repository covers six Course Outcomes (COs) and demonstrates the practical application of advanced data structures and algorithms through real-world case studies.

---

## Course Outcomes

### CO1 – Rank Augmented AVL Tree

**Case Study:** QuizClash Live Tournament Leaderboard

Topics Covered:

* AVL Tree
* Self Balancing BST
* Rank Queries
* Size Augmentation
* O(log n) Updates and Searches

File:

* `RankAVL.java`
